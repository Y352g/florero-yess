    Enfoque del estudio

    El desarrollo de este proyecto se realizó bajo un enfoque práctico con características de tipo descriptivo, ya que me enfoqué en representar el funcionamiento de un sistema de comercio electrónico básico.

    Tipo de investigación

    Considero que el enfoque de investigación es cualitativo, ya que se analizó principalmente la experiencia del usuario y la interacción con el sistema dentro de un entorno simulado.

    Transformación digital

    A través de este proyecto puedo evidenciar cómo la transformación digital permite modernizar procesos tradicionales, facilitando la compra de productos mediante plataformas en línea.

    Uso de tecnologías

    El uso de tecnologías web como HTML, CSS y JavaScript permitió desarrollar un sistema funcional e interactivo, demostrando cómo es posible crear soluciones digitales accesibles y eficientes.
