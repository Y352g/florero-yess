Tecnologías utilizadas

En esta sección del sitio implementé el uso de HTML, CSS y JavaScript para mostrar el catálogo de productos de manera organizada. Considero que el uso de estas tecnologías permite presentar la información de forma clara y atractiva para el usuario.

Almacenamiento

Para simular el funcionamiento de un carrito de compras, utilicé localStorage, lo cual me permitió guardar temporalmente los productos seleccionados sin necesidad de una base de datos real. Esto facilita la persistencia de la información durante la navegación.

Entorno de desarrollo

El proyecto lo desarrollé en Visual Studio Code, ya que facilita la organización y edición del código.

Importancia tecnológica

Desde mi punto de vista, la implementación de este tipo de soluciones tecnológicas permite mejorar la eficiencia de los procesos comerciales y brindar una mejor experiencia al usuario en entornos digitales.
