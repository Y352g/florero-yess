Tipo de e-commerce

B2C (Business to Consumer), ya que el negocio vende directamente al cliente final.

Introducción / Descripción del sitio

En este proyecto desarrollé un prototipo de comercio electrónico llamado “El Florero de Yess”, enfocado en la venta de flores en línea. Mi objetivo fue diseñar un sitio web visualmente atractivo, funcional y fácil de usar, que permita simular una experiencia real de compra digital.

Transformación digital

Desde mi perspectiva, este proyecto representa un ejemplo claro de transformación digital, ya que un negocio tradicional como la venta de flores puede adaptarse al entorno tecnológico mediante el uso de plataformas web. Esto permite ampliar el alcance del negocio y ofrecer una mejor experiencia al cliente.

Digitalización

Considero que la digitalización es un elemento clave en este sistema, ya que procesos que normalmente se realizan de forma manual, como la selección de productos o el registro de pedidos, se realizan aquí de manera automatizada mediante el sitio web.

Uso de TI

Para el desarrollo del sitio utilicé tecnologías web como HTML5 para la estructura, CSS3 para el diseño visual y JavaScript para la interacción del usuario. Estas herramientas permiten crear aplicaciones dinámicas sin necesidad de sistemas complejos en el servidor.

Transformación Digital - Reflexión sobre el sistema

En este proyecto implementé un flujo completo de compra que simula el funcionamiento de un sistema real de comercio electrónico. Integré el uso de almacenamiento temporal mediante localStorage y desarrollé formularios interactivos, lo que me permitió aplicar tecnologías de la información para automatizar procesos comerciales.

Desde mi perspectiva, este sitio demuestra cómo un negocio tradicional puede evolucionar a través de la transformación digital, permitiendo a los clientes realizar compras en línea, mejorar su experiencia y ampliar el alcance del mercado.
